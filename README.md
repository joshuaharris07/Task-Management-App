Task Management Application
A web-based task management application designed to help users organize their tasks, manage priorities, and achieve their goals. This application features an interactive interface for creating, editing, and managing tasks.

Features
Sticky Navbar: Navigation links for quick access to various sections of the app.
Priority Tasks: Display and manage tasks marked as high-priority.
Task Dashboard: Interactive dashboard for viewing, editing, and deleting tasks.
Task Details: Add or update task descriptions, priorities, and due dates.
User Authentication: Register and log in using email and password.
Responsive Design: Set up for desktop or mobile use using Bootstrap
